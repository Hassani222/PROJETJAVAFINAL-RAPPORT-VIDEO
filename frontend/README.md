# Emotional Guard V2 - Interface Frontend

Cette interface est un tableau de bord moderne et responsive conçu pour la gestion des alertes émotionnelles et des utilisateurs.

## Fonctionnalités
- **Dashboard Overview** : Vue d'ensemble des statistiques clés.
- **Gestion des Alertes (CRUD)** : Visualisation, ajout et suppression des alertes détectées par l'IA.
- **Gestion des Utilisateurs (CRUD)** : Administration des comptes psychologues et admins.
- **Design Premium** : Mode sombre, effets de flou (glassmorphism), et animations fluides.
- **Responsive** : Adapté aux mobiles, tablettes et ordinateurs.

## Comment l'ouvrir
1. Naviguez vers le dossier `frontend/`.
2. Ouvrez le fichier `index.html` dans votre navigateur (Chrome, Edge, Firefox).

## Technologies Utilisées
- HTML5 Sémantique
- Vanilla CSS (Design system personnalisé)
- Vanilla JavaScript (Logique dynamique)
- Google Fonts (Outfit)
